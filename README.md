本项目专为一加ACE5设备设计，提供基于Magisk的系统温控与充电策略修改方案，通过优化底层配置文件，实现性能释放与续航平衡的动态调节；同时支持自定义参数，适配不同用户的使用场景（游戏、日常、续航优先）。
 
- 核心特性1：无需修改系统分区，通过Magisk/KSU模块加载，安全无残留
​
- 核心特性2：支持多场景模式切换（如游戏模式、节能模式）
​
- 核心特性3：适配Android 14+系统，兼容一加ACE5官方及第三方ROM
​
- 核心特性4：提供Shell脚本工具，支持命令行快速配置
 
安装指南
 
前置要求
 
1. 设备已解锁Bootloader
​
2. 已安装Magisk 26.0+ 或 KernelSU 0.7.0+
​
3. 系统版本：Android 14及以上（一加ACE5专属适配）
​
4. （可选）已安装Termux（用于命令行配置）
 
安装步骤
 
1. 下载项目最新发布的模块包（[Release页面链接]）
​
2. 打开Magisk/KSU应用，进入「模块」页面
​
3. 点击「从本地安装」，选择下载的模块包（.zip格式）
​
4. 重启设备，模块自动生效
​
5. （可选）通过Termux执行  su -c [脚本命令]  进行自定义配置
 
使用说明
 
基础使用
 
- 模块安装后默认启用「平衡模式」，无需额外配置即可使用
​
- 如需切换模式，可通过以下方式：
​
1. 手动修改配置文件： /data/adb/modules/[模块名]/config.conf 
​
2. 执行Shell脚本： su -c sh /data/adb/modules/[模块名]/script/switch_mode.sh [mode] （mode可选：game/balance/power）

​
2. 手机QQ：进入个人头像→QQ会员→靓号，选号后开通对应会员并支付。
​
3. 电脑QQ：登录后点顶部“靓号”图标，进入靓号站选购支付。
 
二、购买流程（通用）
 
1. 登录账号，浏览/搜索心仪靓号（普通/精品/顶级，价格与会员要求不同） 。
​
2. 普通靓号：付初选费+开通指定时长会员，到期后可续可不续，无“靓”标 。
​
3. 精品/顶级靓号：付初选费+开会员，到期必须续会员才能正常用，带“靓”标 。
​
4. 选开通时长，用Q币/Q点、微信、银行卡等支付 。
​
5. 按提示设置密码、绑定密保手机等，完成后即可使用。
