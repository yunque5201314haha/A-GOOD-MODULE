安卓性能优化工具箱 (Android Performance Tuner)
 
一个专注于安卓设备深度性能调校的Magisk/KSU模块，通过系统级优化提升设备流畅度与游戏体验。
 
核心特性
 
- CPU/GPU智能调度：动态调整核心频率，平衡性能与功耗
- 温控优化：智能温控策略，减少游戏/高负载场景降频
- 内存管理增强：优化后台进程回收机制，提升前台应用响应速度
- IO性能提升：调整磁盘I/O调度策略，加快应用启动与文件读写
- 游戏专属优化：针对《使命召唤手游》等主流游戏的帧率稳定性优化
- 后台守护机制：支持定时任务与进程保活，确保优化策略持续生效
 
支持设备
 
- 理论兼容所有基于ARM架构的安卓设备
- 已深度适配：一加ACE系列（含ACE5）
- 要求：已获取Root权限（Magisk/KernelSU环境）
 
安装说明
 
1. 下载最新版本模块包
2. 在Magisk/KSU管理器中刷入模块
3. 重启设备，优化自动生效
 
使用方法
 
模块安装后默认自动运行，无需额外配置。
如需自定义参数，可编辑  /data/adb/modules/performance_tweaker/config.sh  文件。
