# 安卓性能优化脚本
基于Shell/Magisk的安卓设备性能调控工具，支持CPU/GPU频率、温控、内存优化，适配一加ACE5等机型。

## 功能特性
- CPU/GPU频率动态调控，提升游戏/日常性能
- 温控配置自定义，降低发热与降频
- 开机自启+后台守护，稳定运行不被杀
- 兼容Magisk/KSU，支持一键刷入

## 快速开始
### 安装
1. 下载最新release包
2. Magisk/KSU中刷入模块
3. 重启设备生效

### 使用命令
```bash
# 查看当前状态
./performance.sh status
# 开启性能模式
./performance.sh mode performance
# 恢复默认
./performance.sh reset
