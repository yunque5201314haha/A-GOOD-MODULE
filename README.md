项目名称
 
安卓性能优化工具箱 (Android Performance Tuner)
 
项目简介
 
本项目为一加ACE5等安卓设备提供深度性能定制方案，通过Magisk/KSU模块、Shell脚本与系统配置优化，实现CPU/GPU调度、温控调节、内存管理等底层参数调整，提升设备运行效率与游戏帧率稳定性。
 
核心功能
 
- CPU/GPU性能调控：自定义频率阈值、调度策略，平衡性能与功耗
- 温控优化：修改Thermal配置文件，解除过度温控限制
- 内存管理：优化ZRAM参数、调整后台进程限制，提升多任务流畅度
- 脚本自动化：提供开机自启、定时执行的Shell脚本，实现参数持久化
- 系统修复：包含谷歌服务清理、刷机故障修复等实用脚本
 
支持设备
 
- 一加ACE5 (主设备)
- 理论兼容其他高通/联发科平台安卓设备 (需适配参数)
 
安装要求
 
- 已获取Root权限 (Magisk/KernelSU)
- 安装Busybox工具链
- 安卓版本：Android 11及以上
 
安装步骤
 
1. 下载最新Release版本模块包
2. 在Magisk/KSU中刷入模块
3. 重启设备，模块自动生效
4. 可通过终端执行 /data/adb/modules/performance_tuner/script.sh 手动调整参数
 
配置说明
 
- 核心配置文件： /data/adb/modules/performance_tuner/config.conf 
- 支持自定义CPU频率上限、GPU频率档位、温控阈值等参数
- 提供性能模式、均衡模式、省电模式三种预设方案
 
注意事项
 
- 刷机与修改系统参数存在风险，操作前请备份数据
- 过度超频可能导致设备发热、死机或硬件损坏
- 部分参数修改需结合设备具体配置，建议先测试再长期使用
