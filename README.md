🚀 安装指南
 
1. 下载仓库内「[模块名称].zip」压缩包（或Release页最新版本）；
2. 打开Magisk/KSU管理器，进入「模块」页面，点击「安装模块」；
3. 选择下载的压缩包，等待刷入完成后重启设备；
4. 验证生效：
- 用CPU-Z查看超大核频率（应显示3.3GHz）；
- 或执行命令： su && cat /sys/devices/system/cpu/cpu0/cpufreq/scaling_cur_freq 
 
⚙️ 自定义配置（可选）
 
若需适配其他机型或调整参数，可按以下步骤修改：
 
1. 解压模块包，编辑「service.sh」文件；
2. 修改cpu编号（如非骁龙8 Gen3机型，需确认超大核对应cpu序号）；
3. 调整最高频率值（单位：kHz，如骁龙8 Gen3为3300000）；
4. 重新压缩为zip格式，刷入模块并重启。
 
⚠️ 重要提醒
 
1. 操作前请备份系统数据，避免适配问题导致异常；
2. 锁定最高频会增加功耗与发热，建议仅高负载场景使用，日常可卸载模块；
3. 若出现过热降频/系统卡顿：
- 卸载模块并重启；
- 或执行恢复命令： su && echo schedutil > /sys/devices/system/cpu/cpu0/cpufreq/scaling_governor
