本项目专为一加ACE5设备设计，提供基于Magisk的系统温控与充电策略修改方案，通过优化底层配置文件，实现性能释放与续航平衡的动态调节；同时支持自定义参数，适配不同用户的使用场景（游戏、日常、续航优先）。
 
- 核心特性1：无需修改系统分区，通过Magisk/KSU模块加载，安全无残留
​
- 核心特性2：支持多场景模式切换（如游戏模式、节能模式）
​
- 核心特性3：适配Android 14+系统，兼容一加ACE5官方及第三方ROM
​
- 核心特性4：提供Shell脚本工具，支持命令行快速配置
 
安装指南
 
前置要求
 
1. 设备已解锁Bootloader
​
2. 已安装Magisk 26.0+ 或 KernelSU 0.7.0+
​
3. 系统版本：Android 14及以上（一加ACE5专属适配）
​
4. （可选）已安装Termux（用于命令行配置）
 
安装步骤
 
1. 下载项目最新发布的模块包（[Release页面链接]）
​
2. 打开Magisk/KSU应用，进入「模块」页面
​
3. 点击「从本地安装」，选择下载的模块包（.zip格式）
​
4. 重启设备，模块自动生效
​
5. （可选）通过Termux执行  su -c [脚本命令]  进行自定义配置
 
使用说明
 
基础使用
 
- 模块安装后默认启用「平衡模式」，无需额外配置即可使用
​
- 如需切换模式，可通过以下方式：
​
1. 手动修改配置文件： /data/adb/modules/[模块名]/config.conf 
​
2. 执行Shell脚本： su -c sh /data/adb/modules/[模块名]/script/switch_mode.sh [mode] （mode可选：game/balance/power）一加ACE5 深度优化工具箱
 
基于 Magisk/KSU 的一加ACE5 系统级优化模块，专注性能释放、功耗控制与功能增强
 
模块特性
 
- 性能调度优化：动态调整CPU/GPU频率，平衡游戏帧率与日常功耗
- 系统底层优化：精简后台服务、优化IO调度、提升系统流畅度
- 功能增强：支持自定义刷新率、温控阈值、触控灵敏度
- 安全稳定：无系统修改残留，支持一键还原，兼容最新ColorOS版本
- 轻量化脚本：Shell脚本加密混淆，执行高效，占用资源极低
 
支持设备
 
- 一加ACE5（全版本/全系统版本）
 
安装要求
 
- 已获取Root权限（Magisk 26.0+ / KernelSU 0.7.0+）
- 剩余存储空间 ≥ 50MB
 
安装步骤
 
1. 下载最新模块压缩包
2. 打开Magisk/KSU Manager → 模块 → 从本地安装
3. 刷入模块后重启设备
4. 重启后通过终端执行 ace5-opt 调用配置工具
 
常用命令
 
bash  
# 打开配置菜单
ace5-opt

# 一键性能模式
ace5-opt --mode performance

# 一键省电模式
ace5-opt --mode battery

# 还原默认设置
ace5-opt --reset

1. 刷机/系统更新前建议先卸载模块，避免兼容性问题
2. 若出现异常，可通过Recovery挂载系统后删除 /data/adb/modules/ace5_optimize 目录还原
3. 本模块仅适用于一加ACE5，其他设备请勿刷入
